# CHANGELOG — Админ-панель CursorRandka

## Новая админ-панель (Telegram)

### Вход
- Команда `/admin` — главное меню с inline-кнопками (русский интерфейс)
- Не-админ получает «Доступ запрещён»

### Разделы панели
- **📊 Статистика** — пользователи, активность за 24ч, профили, лайки, матчи, жалобы, регистрации, топ-5 городов
- **👥 Пользователи** — последние 10 с пагинацией, карточка пользователя
- **🚨 Жалобы** — pending-жалобы с ban / shadow ban / отклонить
- **🔍 Найти юзера** — поиск по tg_id, @username или имени
- **📢 Рассылка** — предпросмотр, подтверждение, лимит ~30 сообщ/сек

### Действия с пользователем
- 🚫 Ban / ✅ Unban
- 👻 Shadow ban (скрыт из ленты, пользователь не уведомляется)
- 🗑️ Полное удаление аккаунта
- 📊 Статистика (лайки, матчи, просмотры)

### Команды
- `/admin` — панель
- `/stats` — быстрая статистика
- `/ban <tg_id> [причина]`
- `/unban <tg_id>`
- `/user <tg_id>` — быстрый просмотр

### База данных
- `is_shadow_banned`, `ban_reason`, `last_active` в `users`
- `reports.status` — pending / dismissed / actioned
- Таблица `admin_log` — аудит действий админа
- Кандidate feed исключает banned и shadow_banned

### Безопасность
- Единая проверка `is_admin()` / `@admin_only`
- Все callback с префиксом `adm:`
- Параметризованные SQL-запросы
- Логирование в БД и logger

### Запуск
Перезапустите бота через `start.bat`, затем отправьте `/admin` в Telegram.
