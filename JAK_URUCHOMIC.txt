CURSORRANDKA — КАК ЗАПУСТИТЬ БОТА
==================================

ШАГ 1. Установите Python 3 (если ещё не установлен)
   Скачайте с https://www.python.org/downloads/
   При установке обязательно отметьте галочку «Add Python to PATH».

ШАГ 2. Вставьте токен бота в файл .env
   ─────────────────────────────────────
   Откройте ЭТОТ файл в Блокноте (Notepad):

      C:\Users\nazar\OneDrive\Рабочий стол\CursorRandka\.env

   Найдите ЭТУ строку (первая строка файла):

      BOT_TOKEN=your_bot_token_here

   Замените your_bot_token_here на ваш настоящий токен.
   Пример после замены:

      BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

   Сохраните файл (Ctrl+S) и закройте Блокнот.

   Как получить токен:
   1. Откройте Telegram.
   2. Найдите @BotFather.
   3. Отправьте команду /newbot (или /mybots → выберите бота → API Token).
   4. Скопируйте выданный токен и вставьте вместо your_bot_token_here.

   ADMIN_ID уже настроен — менять не нужно.

ШАГ 3. Запустите бота
   ───────────────────
   Дважды щёлкните по файлу:

      C:\Users\nazar\OneDrive\Рабочий стол\CursorRandka\start.bat

   Откроется чёрное окно — бот работает, пока окно открыто.
   Чтобы остановить бота, закройте это окно.

ПРОБЛЕМЫ?
─────────
• «Brak BOT_TOKEN» или «BOT_TOKEN nie jest ustawiony»
  → Вы не заменили your_bot_token_here в файле .env (см. Шаг 2).

• «python не является внутренней или внешней командой»
  → Переустановите Python с галочкой «Add to PATH».

• Бот не отвечает в Telegram
  → Проверьте, что токен скопирован полностью, без пробелов.
  → Убедитесь, что окно start.bat открыто и в нём нет ошибок.
