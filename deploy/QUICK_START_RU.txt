ЧТО КУПИТЬ:
  Hetzner CX22 — ~20 zł/мес (рекомендуется)
  или Oracle Cloud Free — 0 zł (сложнее)

───────────────────────────────────────────────────

ШАГ 1. Создать VPS
  hetzner.com → CX22 → Ubuntu 22.04 → записать IP

ШАГ 2. Подготовить .env
  Скопировать .env.example → .env
  Вписать BOT_TOKEN и ADMIN_ID

ШАГ 3. Загрузить на сервер
  Открыть deploy/upload_from_windows.ps1
  Заменить YOUR_SERVER_IP → запустить скрипт

ШАГ 4. Установить (на сервере)
  ssh root@ВАШ_IP
  cd /opt/randkapl && sudo bash deploy/install.sh

ШАГ 5. Проверить
  systemctl status randkapl
  Написать боту /start в Telegram

───────────────────────────────────────────────────

ГОТОВО! Бот работает 24/7.
